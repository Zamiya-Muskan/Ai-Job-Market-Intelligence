## 📌 Project Overview

The **AI Job Market Intelligence Dashboard** is an end-to-end data analytics and forecasting project that analyzes AI job postings collected between **2024 and 2026**. The project provides insights into hiring trends, skill demand, job roles, industries, salaries, and geographical distribution while forecasting future hiring demand using **XGBoost**.

The dashboard is designed to help students, job seekers, recruiters, and organizations understand current market trends and make data-driven career and hiring decisions.

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/Zamiya-Muskan/AI-Job-Market-Intelligence.git
```

Move into the project directory:

```bash
cd AI-Job-Market-Intelligence
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Streamlit application:

```bash
streamlit run app.py
```
