# 🤖 AI Job Market Intelligence Dashboard

An interactive **AI Job Market Analytics & Forecasting Dashboard** built using **Python, Streamlit, Plotly, XGBoost, and Statistical Analysis**.

The dashboard analyzes AI job postings collected between **2024–2026** to uncover hiring trends, salary insights, in-demand skills, emerging technologies, and future hiring demand. It also includes statistical validation and an AI Skill Gap Analyzer to help users evaluate their career readiness.

# 📌 Project Overview

The rapid growth of Artificial Intelligence has transformed the job market, creating demand for new technical skills and specialized roles.

This project provides an interactive platform that enables users to:

- Analyze AI hiring trends
- Explore salary distributions
- Identify in-demand skills
- Monitor emerging AI technologies
- Forecast future hiring demand
- Compare personal skills with market requirements

The dashboard combines data analytics, statistics, machine learning, and visualization to support informed career and business decisions.

# 📈 Key Insights

- Python remains the most demanded programming language for AI roles.
- AI hiring remained consistently strong throughout 2025.
- Cloud technologies, SQL, and REST APIs continue to be highly valued.
- Emerging technologies such as LangChain, OpenAI API, RAG, and Hugging Face show increasing demand.
- Experience level has a statistically significant impact on salary.
- The forecast indicates continued demand for AI professionals during the next six months.

# ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/Zamiya-Muskan/AI-Job-Market-Intelligence.git
```

Navigate to the project directory:

```bash
cd AI-Job-Market-Intelligence
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
